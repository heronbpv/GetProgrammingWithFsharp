Quick check 12.1 answers:
1 - Can you store values in namespaces?
A - No, only type declarations.

2 - Can you store types in modules?
A - Yes.

Quick check 12.2 answers:
1 - In what order are files read for dependencies in F#?
A - Top to bottom.

2 - When can you omit a module declaration in an F# file?
A - When the file is the last one in the project definition.

Quick Check 12.3 answers:
1 - What is the AutoOpen attribute for?
A - To make the modules inside a namespace, which are decorated with this attribute, be automatically opened as well.

2 - What is the default access modifier for values in modules?
A - Public.
