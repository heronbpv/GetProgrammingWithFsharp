Quick check 13.1 answers:
1 - Name one difference between nominal and structural types.
A - Nominal types cannot be implicitly exchanged for one another even if they have the same structure, unlike structural types.

2 - How do we pass logic between or across code in the OO world?
A - Through interfaces.

3 - How do we pass logic between or across code in the FP world?
A - Through functions.

Quick check 13.2 answers:
1 - Can F# infer the types of higher-order functions?
A - Yes.

2 - How can you easily identify higher-order function arguments in VS?
A - By noticing the color of the argument. It's gonna be the same as the color of the function name. Assuming, of course, that the Visual F# Power Tools plugin for Visual Studio 2015 is installed.
